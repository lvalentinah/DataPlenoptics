Microscopio

La microscopia es un proceso de investigación en donde se utiliza un microscopio para obtener imágenes de determinadas estructuras microscópicas, las cuales no son posibles observarlas a simple vista para el ojo humano.[1] Los objetos mínimos que puede observar el ojo humano son de 0.2mm (200µm) para tamaños menores se utiliza un microscopio [2]. El principio de funcionamiento de un microscopio óptico se basa en la propiedad de algunos materiales que permiten cambiar la dirección de los rayos de luz, los cuales son capaces de hacer converger o divergir los rayos de luz y gracias a la combinación de estos lentes se puede generar una imagen aumentada de cualquier objeto, tal y como se muestra

El instrumento más utilizado es el microscopio óptico el cual utiliza un haz de luz (fotones) para amplificar las muestras, el cual está conformado por tres sistemas:

• Sistema mecánico: Este sistema proporciona soporte, movimiento y estabilidad a todos sus componentes (Base o pie, brazo o columna, platina, tubo óptico, revolver, tornillos macrométricos y micrométricos).
• Sistema óptico: Este sistema está encargado de producir el aumento de las imágenes (objetivos y oculares).
• Sistema de iluminación: Corresponde a las partes que reflejan, transmiten y regulan la cantidad de luz necesaria para efectuar la observación a través del microscopio (Condensador, espejo, fuente de iluminación y diafragma).

Existen distintos tipos de microscopios de acuerdo al número de oculares, los cuales son:

• Monocular: Es el tipo de microscopio que dispone de un solo ocular para observar la muestra y para la observación se utiliza un solo ojo.
• Binocular: Es el tipo de microscopio compuesto por dos oculares para observar la muestra, contiene internamente un prisma divisor de luz, el cual desvía la luz hacia dos oculares en un 50% para cada uno
• Trinocular: Es el tipo de microscopio compuesto por 3 oculares, en donde puede utilizarse como microscopio binocular por un observador y simultáneamente se puede conectar en el tercer ocular una cámara para capturar imágenes de la observación. En este microscopio el prisma interno divide el haz de luz en tres haces iguales, lo que permite que la observación sea igual en los tres oculares. Los dos oculares deben estar ubicados con una inclinación de 45° respecto a la superficie horizontal y el tercer ocular se debe ubicar verticalmente en el cabezal del microscopio.
